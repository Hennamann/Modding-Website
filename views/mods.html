My Mods

Mist Core

A small library for adding fog/mist to biomes easily. Creates the same fog effect found in dimensions but in biomes. The mod adds no content on it's own and is intended to be used alongside other mist mods, like Mist Biomes.

Releases CurseForge Project Issues Source Developer Documentation
Mist Biomes

Adds several "mistified" biomes from vanilla Minecraft like Mist Plains, Mist Forest, Mist Deserts. This mist is for the time being mainly cosmetic but the plan is to add both utility and danger to the mist in later versions of the mod.

Releases CurseForge Project Issues Source
Wasteland Reborn

The classic Wasteland mod ported from 1.10.2 to 1.11.2 and 1.12.2 with several improvements and new features and way more configuration options for modpack makers and the like. Makes for a real challenge in Minecraft survival especially if you combine it with mods like Tough as Nails.

Releases CurseForge Project Issues Source
NMCLC (NoMoreCapsLockCommands)

Really simple mod for solving a luxury problem; It converts commands written in uppercase to lowercase for those situations where you unintentionally left the capslock key on when typing in a command.
